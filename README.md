# Insurance Analytics Report

This repository contains an interactive **Power BI** dashboard designed to analyze insurance policy performance, customer metrics, and retention trends.

## 📊 Dashboard Overview

The Power BI report consists of two primary analytical tabs:

### 1. Insurance Customers Analytics
Provides a high-level overview of policy distributions and claims tracking. 
**Key Performance Indicators (KPIs) include:**
* **Premium Amount:** Total financial volume generated from policy premiums.
* **Active/Inactive Policies:** A comparative breakdown of operational vs. terminated policies.
* **No. of Claims:** The total volume of insurance claims filed by policyholders.

### 2. Cohort Analysis
Focuses on long-term behavioral trends and retention tracking.
* **Customer Insurance Cohorts:** Visualizes groups of policyholders over time to analyze lifecycle patterns, renewal rates, and customer longevity.

---

## 🛠️ Tools Used
* **Power BI:** Data modeling, DAX measures, and interactive visual reporting.

## 🚀 How to View the Report
1. Download the `Insurance_Analysis.pbix` file from this repository.
2. Open the file using [Power BI Desktop](https://powerbi.microsoft.com/desktop/) to interact with the filters and dashboards.
